# IceCity Heating System 🔥

## 📌 Overview
A C# console application that simulates a heating management system.

## 🧠 Concepts Applied
- OOP (Encapsulation, Inheritance, Polymorphism)
- SOLID Principles
- Strategy Pattern
- Factory Pattern
- Async/Await & Multithreading

## ⚙️ Features
- Calculate heating cost
- Event-driven heater behavior
- Multiple cost strategies (Standard / Eco)
- Exception handling and recovery

## 🏗 Architecture
- Models
- Services
- Strategies
- Factories
